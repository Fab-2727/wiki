# Wiki emulation

This a web application that emulates [Wikipedia](https://en.wikipedia.org/wiki/Main_Page).

## What I learned
  - Django framework
  - Django structure project
  - Handling multiples HTTP methods in the same function
  - Use of Django forms
  - Django template language
  - Handling different types of request.

### Functions available:
  - Search for a specific page
  - Create a new page
  - Update an already existing page
  - Get a random entry
